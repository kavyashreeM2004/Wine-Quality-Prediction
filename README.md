# Wine-Quality-Prediction
Machine Learning project to predict wine quality using EDA, preprocessing, and models.


## Project Overview
This project predicts wine quality using machine learning techniques.

## Dataset
Wine Quality Dataset (Red Wine)

## Steps Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Scaling
- Model Training:
  - Linear Regression
  - Logistic Regression
- Model Evaluation

## Results
- Logistic Regression Accuracy: ~75%
- Linear Regression R² Score: ~0.35

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
